* **Description:** A highly available cloud environment provisioning platform paired with an end-to-end continuous integration and deployment ecosystem.
* **Core DevOps Execution:** 
  * Architected custom AWS VPCs, Security Groups, Jenkins Master/Agent topologies, and Ansible Controllers using Terraform.
  * Engineered a multi-branch Jenkins pipeline integrated with GitHub webhooks, forcing strict code compilation through SonarQube code quality gates.
  * Containerized the stack via Docker, archived versioned artifacts in JFrog Artifactory, and managed rolling zero-downtime updates to AWS EKS cluster topologies.

* **Tech Stack:** Terraform, Ansible, Jenkins, SonarQube, JFrog Artifactory, Docker, Kubernetes, AWS EKS, Prometheus, Grafana, Helm

* **Description:** A rigid, automated quality gate and container security pipeline deployed to isolate, audit, and push an enterprise full-stack Spring Boot blogging platform.
* **Core DevOps Execution:**
  * Orchestrated multi-stage, containerized build agents executing Unit, Integration, and End-to-End (E2E) suites concurrently via JUnit, Jest, and Selenium.
  * Implemented strict Shift-Left security by running deep static code analysis via SonarQube and container image vulnerability scanning via Trivy directly inside the pipeline stages.
  * Handled safe artifact version management via Nexus Repository and established real-time deployment status reporting via transactional email notifications.
* **Tech Stack:** Jenkins, Docker, AWS EKS, SonarQube, Trivy, JUnit, Jest, Selenium, Nexus Repository
